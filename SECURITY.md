## Reporting a Vulnerability

We take the security of our software seriously. If you believe you have found a security vulnerability in the **WorkLedger Hub**, please report it to us as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

### Procedure

1.  **Email**: Send an email to [arunkumarbind150720@gmail.com] with a description of the vulnerability.
2.  **Confirmation**: We will acknowledge your email within 48 hours.
3.  **Investigation**: We will investigate the issue and communicate with you to confirm the details.
4.  **Fix**: Once confirmed, we will work to patch the vulnerability.
5.  **Disclosure**: We will coordinate the public disclosure of the vulnerability with you.

### What to Include

- A step-by-step guide to reproduce the vulnerability.
- The version of the software where you found the issue.
- Any relevant logs or output.

Thank you for helping keep our project safe!
